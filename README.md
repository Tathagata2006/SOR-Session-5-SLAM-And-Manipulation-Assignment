# ROS 2 Navigation and Manipulation Assignment

This repository contains my submission for the ROS 2 Navigation and Manipulation assignment.

## Repository Structure

```
.
├── Submission_Codebase/
├── Q1_Demo_Video.mp4
├── Q2_Demo_Video.mp4
├── Q2_Part_A.pdf
└── README.md
```

### Contents

- **Submission_Codebase/** – Complete ROS 2 workspaces and source code for both questions.
- **Q1_Demo_Video.mp4** – Demonstration of autonomous waypoint navigation.
- **Q2_Demo_Video.mp4** – Demonstration of robotic arm control using custom inverse kinematics.
- **Q2_Part_A.pdf** – Analytical inverse kinematics calculations for the robotic arm.
